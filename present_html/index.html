Installation Guide

Step 1: Prepare a fresh Ubuntu 24.04 LTS server.


Step 2: Run the auto-installer:

# Download & Install wget https://github.com/vietdev99/vhestiacp/raw/main/install/hst-install-ubuntu.sh bash hst-install-ubuntu.sh --interactive yes --email admin@you.com

Note: Installation takes about 15-30 minutes depending on your server speed.

Alternative: Use the Config Generator at vietdev99.github.io/vhestiacp

VHestiaCP Dashboard

Service Management

HAProxy

Accessed via Server > Configure > HAProxy.

  • Dashboard: Real-time traffic stats, backend health status.
  • Configuration: Edit global/defaults/frontend/backend sections visually.
  • Visualize: View traffic flow graphs.

Databases

Server > Configure > Databases.

  • Management: Create/Delete databases and users.
  • Connection: View connection strings for your apps.
  • Security: Manage access lists and binder addresses.

Node.js

Manage PM2 processes globally.

  • Versions: Install/Switch Node.js versions (18, 20, 22).
  • Processes: Start/Stop/Restart PM2 applications.
  • Logs: View stdout/stderr logs directly in the browser.

PM2 User Isolation

  • Mechanism: PM2 instances are spawned specifically under the Linux user (e.g., `admin`).
  • Benefits: No `root` execution. Apps cannot access other users' files.

MongoDB Security

  • Authentication: Enforced per-database. No public access without auth.
  • Isolation: Users are restricted to their own logical databases.
  • Network: Binds to localhost by default for security.
